# 产品需求文档（PRD）

## 「刷游戏」微信小程序 v1.0

---

## 一、产品概述

### 1.1 产品定位

**「刷游戏」** 是一款以短视频上下滑动交互为壳、AI 即时生成可玩小游戏为核的微信小程序。用户像刷抖音一样刷游戏，每个游戏都由 AI 根据模板+参数即时生成，并支持用户在创意工坊中用自然语言自定义游戏。

### 1.2 核心价值

| 维度 | 描述 |
|------|------|
| **用户价值** | 碎片时间不刷视频，刷可玩的轻量游戏；无限内容供给，每个游戏都独一无二 |
| **创作者价值** | 创意工坊降低游戏制作门槛，prompt 即游戏，人人可创作并分享 |
| **平台价值** | AI+游戏+短视频交互模式，开创"刷游戏"新品类 |

### 1.3 目标用户

- **核心用户**：18-35 岁，喜欢短视频交互习惯、轻度游戏爱好者、纪念碑谷美学偏好者
- **创作者用户**：有创意但不会编程的用户，通过 prompt 创造游戏

## 二、核心功能

### 2.1 功能模块总览

```
┌──────────────────────────────────────────────────────┐
│                    「刷游戏」小程序                      │
│                                                       │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌───────┐ │
│  │ 🔄       │  │ 🎨      │  │ 🤖      │  │ 👤    │ │
│  │ 刷游戏流  │  │ 创意工坊  │  │ AI 生成  │  │ 我的   │ │
│  │ Game Feed│  │ Workshop │  │ Generate │  │ Profile│ │
│  └──────────┘  └──────────┘  └──────────┘  └───────┘ │
│                                                       │
│  ┌──────────────────────────────────────────────────┐ │
│  │              游戏运行时引擎(Game Runtime)          │ │
│  │   渲染 · 物理 · 音频 · 输入 · 关卡 · 评分          │ │
│  └──────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────┘
```

### 2.2 功能优先级与描述

#### P0 — AI 生成游戏（预设 Prompt 模板）

| 子功能 | 描述 |
|--------|------|
| **模板选择** | 提供 5-8 种游戏模板（解谜、平台跳跃、躲避、收集、推箱子、迷宫等），用户选择模板 |
| **一键生成** | 系统预设精选 prompt，用户点击即可生成游戏，无需输入 |
| **即时可玩** | 生成完成后自动进入游戏界面，3-5 秒内可玩 |
| **生成预览** | 展示游戏缩略图、类型标签、难度评级、预计时长 |

#### P1 — 创意工坊（用户自定义 Prompt）

| 子功能 | 描述 |
|--------|------|
| **Prompt 输入** | 文本输入框，支持自然语言描述想要的游戏 |
| **创意引导** | 提供 prompt 引导模板（"我想要一个____风格的游戏，玩法是____"） |
| **参数调节** | 高级模式：难度滑块、时长选择、颜色主题、特殊元素开关 |
| **生成历史** | 查看自己生成过的所有游戏，支持重新编辑 prompt 再生成 |
| **发布分享** | 满意的游戏可发布到公共游戏流，供其他用户游玩 |

#### P2 — 刷游戏流（上下滑动切换）

| 子功能 | 描述 |
|--------|------|
| **全屏沉浸** | 单个游戏占满屏幕，上下滑动切换到下一个游戏 |
| **游戏自动播放** | 滑到当前游戏后自动展示玩法 demo（3-5 秒 AI 试玩动画） |
| **点击开始** | 点击屏幕任意位置进入游玩模式 |
| **滑动跳过** | 不喜欢直接滑走，支持左滑"不感兴趣" |
| **推荐算法** | 基于游玩行为、完成率、偏好标签的个性化推荐 |

#### P3 — 社交功能

| 子功能 | 描述 |
|--------|------|
| **点赞** | 双击/点击按钮为游戏点赞 |
| **评论** | 游戏下方评论区 |
| **分享** | 分享到微信聊天/朋友圈，生成带游戏截图的分享卡片 |
| **关注** | 关注创作者，查看其创意工坊作品集 |

#### P4 — 用户系统

| 子功能 | 描述 |
|--------|------|
| **游戏记录** | 游玩历史、完成率、游戏时长统计 |
| **成就系统** | 通关里程碑、创意勋章、连续游玩天数 |
| **等级体系** | 游玩和创作获取经验值，解锁更多模板和高级 prompt 能力 |

#### P5 — 商业化

| 子功能 | 描述 |
|--------|------|
| **激励广告** | 看广告解锁稀有模板、增加每日生成额度 |
| **会员订阅** | 无限生成次数、高级模板、优先推荐 |
| **打赏** | 对喜欢的创作者和游戏进行打赏 |

## 三、用户流程

### 3.1 核心流程：刷游戏

```
打开小程序 → 进入游戏流 → 上下滑动浏览
                              │
                    ┌─────────┴─────────┐
                    ▼                   ▼
              感兴趣 → 点击开始      不感兴趣 → 继续滑动
                    │
                    ▼
              游玩游戏（无限时长，用户自行决定）
                    │
          ┌─────────┼─────────┐
          ▼         ▼         ▼
       通关/放弃   点赞     分享
          │
          ▼
       继续刷下一个
```

### 3.2 核心流程：创作游戏

```
进入创意工坊 → 选择「预设模板」或「自由创作」
                              │
              ┌───────────────┴───────────────┐
              ▼                               ▼
        选择模板 + 预设 prompt           输入自定义 prompt
              │                               │
              └───────────────┬───────────────┘
                              ▼
                      点击「生成游戏」
                              │
                              ▼
                      等待 3-10 秒 AI 生成
                              │
                              ▼
                      预览 + 试玩游戏
                              │
                    ┌─────────┴─────────┐
                    ▼                   ▼
              满意 → 发布/分享     不满意 → 修改 prompt
                                                │
                                                ▼
                                          重新生成
```

### 3.3 游戏时长分级

| 类型 | 时长 | 适用场景 | 生成策略 |
|------|------|---------|---------|
| **快游戏** | 15-60秒 | 碎片时间、通勤 | 单关卡、简单机制 |
| **短游戏** | 1-5分钟 | 排队等待 | 3-5 关、渐进难度 |
| **长游戏** | 5-30分钟 | 休闲放松 | 多关卡、完整剧情、存档 |

## 四、MVP 范围与里程碑

### Sprint 1: 渲染引擎核心（2 周）

- [ ] WebGL 等距渲染器 MVP（几何体 + 着色 + 交互）
- [ ] Game DSL Parser（JSON → 可渲染场景）

### Sprint 2: AI 生成管道（2 周）

- [ ] 5 个基础游戏模板
- [ ] AI 生成 API 端到端（prompt → DSL → 可玩游戏）
- [ ] 后端服务搭建

### Sprint 3: 游戏流 + 创意工坊（2 周）

- [ ] 全屏滑动游戏流
- [ ] 创意工坊 UI + prompt 输入
- [ ] 预设 prompt 模板（20 个一键生成）

### Sprint 4: 社交 + 上架（1 周）

- [ ] 点赞/分享/推荐
- [ ] 性能优化（首屏 < 2s，帧率 > 30fps）
- [ ] 内容安全审核接入
